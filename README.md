# Conceptual Sessions Code

এই রিপোজিটরিতে আমাদের কোর্সের **Conceptual Session** গুলোর কোড সংরক্ষণ করা হয়েছে।  
আমাদের মূল কোর্সের প্রতি সপ্তাহে সাধারণত **দুটি Conceptual Session** হয়।  

এই সেশনগুলোতে আমরা মূলতঃ  
- রিলিজকৃত মডিউলের গুরুত্বপূর্ণ ও অতিরিক্ত বিষয়গুলো বিস্তারিতভাবে আলোচনা করি,  
- এবং সেই টপিকের উপর ভিত্তি করে **Problem Solving** সেশন করি।
